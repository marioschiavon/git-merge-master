# 09. Listas

**Quando usar:** para agrupar leads por campanha/ICP/importação.
**Pré-requisitos:** [08](./08-leads.md).

## O que é

Listas são "pastas" de leads. Um lead pertence a **uma lista** (ou nenhuma). Cadências podem ser vinculadas a uma lista, e novos leads que entram nela caem automaticamente na cadência ligada.

## Passo a passo

1. Vá em **Listas → Nova lista**.
2. Nome (ex.: "SP Beleza Q1"), descrição opcional.
3. **Cadência padrão** (opcional): se preencher, leads dessa lista com enrichment concluído entram automaticamente na cadência.
4. Salve.

## Dicas

- Uma lista por **campanha** ou **ICP**, não por status. Status já existe no lead.
- Se você importa CSV mensalmente, mantenha uma lista por mês para métricas claras.
