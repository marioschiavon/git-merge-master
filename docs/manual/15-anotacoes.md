# 15. Anotações

**Quando usar:** para registrar aprendizados sobre um lead ou marcar uma decisão que a IA deve respeitar depois.
**Pré-requisitos:** ter leads em conversa.

## O que é

Anotações internas por lead (não aparecem para o cliente). A IA lê essas anotações antes de gerar próximas mensagens — então uma anotação como "não é o decisor, foco no Bruno" muda o comportamento.

## Passo a passo

1. Abra **Anotações** (ou o card de anotação dentro de um lead).
2. Escreva a nota. Marque:
   - **Aplica ao lead** — só para esse contato.
   - **Aplica à empresa toda** — vira contexto para outros contatos da mesma empresa.
3. Salvar.

## Boas notas

- "Cliente pediu para não contatar até fev/26."
- "Empresa comprou concorrente — abordar como upsell."
- "Prefere WhatsApp, ignora email."
