# 03c. Apollo

**Rota:** `/settings/integrations` → **Apollo**
**Quando usar:** para buscar leads B2B com dados de LinkedIn e telefone.
**Pré-requisitos:** conta Apollo com API key ativa.

## O que é

Apollo é a principal fonte de leads da maioria dos nossos clientes. Ele fornece nome, cargo, empresa, LinkedIn, email e telefone. A partir do Apollo você faz buscas por segmento, cargo, região e importa em massa para o Leaderei.

## Passo a passo

1. No Apollo, vá em **Settings → API** e gere/copie sua **API key**.
2. No Leaderei: **Configurações → Integrações → Apollo → Conectar**.
3. Cole a API key e salve.
4. Status vira **Ativo**.

## Dicas

- A API key do Apollo é sensível — quem tiver acesso pode fazer buscas em seu nome. Não compartilhe.
- Se sua conta Apollo tem limite mensal, monitore consumo no painel Apollo.
