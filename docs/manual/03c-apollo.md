# 03c. Apollo

**Quando usar:** você quer buscar leads B2B com dados prontos (nome, cargo, empresa, LinkedIn, email e telefone).
**Pré-requisitos:** conta paga no Apollo com acesso à **API**.

## O que é

Apollo é a maior base de dados B2B do mercado — a principal fonte de leads da maioria dos nossos clientes. Depois de conectar, você usa o app do Leaderei para fazer buscas por segmento, cargo, região e importar centenas de leads de uma vez, direto para as suas listas.

## Passo a passo

### 1. Gerar a API key dentro do Apollo

Uma **API key** é uma "senha longa" que autoriza o Leaderei a fazer buscas em sua conta Apollo.

No Apollo:

1. Clique na **sua foto**, no canto superior direito.
2. Vá em **Settings**.
3. No menu à esquerda, clique em **Integrations**.
4. Procure o card **API** e clique **Configure**.
5. Clique **Create new key** (ou copie uma existente).
6. Copie a chave gerada (ela começa com letras/números). **Guarde bem** — o Apollo só mostra uma vez.

### 2. Colar no Leaderei

1. **Configurações → Integrações → Apollo** → **Conectar**.
2. Cole a API key no campo indicado.
3. Salve.

O status vira **Ativo ✅** imediatamente se a chave for válida.

## Dicas

- A API key é **sensível**. Quem tiver ela pode fazer buscas em seu nome e consumir seus créditos. Nunca compartilhe por email/WhatsApp.
- Se sua conta Apollo tem limite mensal de exports, acompanhe o consumo no próprio painel do Apollo.
- Se a chave parar de funcionar (revogada, conta expirada), o Leaderei mostra **Erro**. Basta gerar nova chave e colar de novo.
