# Manual do Leaderei

Bem-vindo(a) ao Leaderei — sua plataforma de prospecção B2B com IA.

Este manual está organizado na **ordem que um novo usuário deve seguir**. Cada arquivo é uma etapa curta e independente. Se você acabou de entrar, comece pelo [00 — Primeiros passos](./00-primeiros-passos.md) e vá seguindo o rodapé "Próximo passo →" de cada página.

## Fluxo recomendado

```text
Fase 1 — Configuração inicial
  01 Configurações gerais      → 02 Equipe      → 03 Integrações
  ├─ 03a WhatsApp (Hook7)
  ├─ 03b Email (domínio)
  ├─ 03c Apollo
  ├─ 03d Pipedrive (opcional)
  └─ 03e Cal.com
  04 Base de Conhecimento      → 05 Scripts IA  → 06 Intents & Ações

Fase 2 — Operação
  07 Buscar Apollo             → 08 Leads       → 09 Listas
  10 Cadências                 → 11 Aprovações  → 12 Acompanhamento

Fase 3 — Relacionamento
  13 Conversas → 14 Inbox humana → 15 Anotações → 16 Reuniões

Fase 4 — Análise
  17 Dashboard → 18 Relatórios → 19 Runs do Agente
```
