# 17. Dashboard

**Rota:** `/dashboard`
**Quando usar:** visão executiva rápida, todo dia.
**Pré-requisitos:** operação rodando.

## O que é

Tela inicial com os KPIs principais dos últimos 7/30 dias:
- Leads novos.
- Enrichment concluído / falhado.
- Mensagens enviadas (por canal).
- Taxa de resposta.
- Reuniões agendadas / realizadas.
- Funil por status.

## Como usar

- Compare **7 dias vs 30 dias** para pegar tendências.
- Se **enrichment failed > 20%**, revise credenciais (Apify, Apollo).
- Se **resposta cai semana a semana**, sua mensagem está saturando — troque script.
